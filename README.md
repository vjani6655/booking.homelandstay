# Homeland - Property Management System

A modern, minimalist property management system for Airbnb business.

## Features
- Booking management
- Calendar view
- Revenue analytics
- Customer-facing booking form
- Notification system
- Guest rating system

## Setup
1. Start PHP server: `php -S localhost:8000`
2. Default login: admin / admin123
3. Access customer form: `/booking.html`

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP 7.4+
- Database: SQLite
- Security: CSRF tokens, password hashing, session management
